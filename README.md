<h2 id="run">Run Locally</h2>

- Clone the project

```bash
  git clone https://github.com/Vikram-Hegde/bibliophyle
```

- Go to the project directory

```bash
  cd bibliophyle
```

- Install dependencies

```bash
  [p]npm install
```

- Start the server

```bash
  [p]npm dev
```
